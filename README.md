## Student Registration & Validation System ##
demo :https://alienq7.github.io/fswdpract2
# Tech Stack #
Frontend: HTML5, CSS3
Logic: JavaScript (ES6)
Styling: Custom CSS for dynamic error states
# Key Features #
Form Interception: Uses preventDefault() to validate data without page reloads.

Dynamic Validation: * Name: Checks for non-empty strings.

Email: Verifies the presence of the @ symbol.

Password: Enforces a minimum length of 6 characters.

Real-time Feedback: Displays specific error messages in red directly below the invalid input field.
